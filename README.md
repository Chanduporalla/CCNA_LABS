# CCNA Networking Labs

This repository contains my **hands-on CCNA practice labs** built using **Cisco Packet Tracer**. The labs focus on fundamental networking concepts such as routing, switching, IP addressing, VLAN configuration, remote access, and troubleshooting.

The goal of this repository is to strengthen practical networking knowledge and demonstrate real-world configuration skills required for **Network Engineering roles and CCNA certification**.

---

## 📌 Objectives

* Practice **Cisco networking configurations**
* Understand **routing and switching concepts**
* Gain experience with **network troubleshooting**
* Prepare for the **Cisco CCNA certification**
* Build practical skills used by **network engineers**

---

## 🛠 Tools Used

* Cisco Packet Tracer
* Command Line Interface (CLI)
* Basic networking utilities (Ping, Telnet, SSH)

---

## 📚 Lab Topics Covered

### 1. IP Addressing and Subnetting

* IPv4 addressing
* Subnetting practice
* Network and host calculations

### 2. Switching Fundamentals

* Switch configuration
* MAC address table
* Port configuration

### 3. VLAN Configuration

* Creating VLANs
* Assigning ports to VLANs
* Inter-VLAN communication basics

### 4. Routing Configuration

* Static routing
* Basic router configuration
* Default routes

### 5. Remote Access Configuration

* Telnet configuration
* SSH configuration
* Secure remote management

### 6. Network Troubleshooting

* Using ping and traceroute
* Interface status checks
* Diagnosing connectivity issues

---

## 💻 Sample Cisco Commands Used

```bash
enable
configure terminal
interface fastEthernet0/1
ip address 192.168.1.1 255.255.255.0
no shutdown
exit
show ip interface brief
show running-config
```
